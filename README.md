# Watch nhso token

[WatchNhosToken](https://dixonsatit.github.io/watch-nhso-token) เป็น aplication เล็กๆ ไว้ตรวจสอบการเปลี่ยนแปลงของไฟล์ token  สปสช. เพื่อนำข้อมูล token ไปใช้ในกรณีต่างๆ ได้แบบ realtime โดยส่งข้อมูลผ่าน MQTT
